# 社团管理系统 java667

## 项目概述

本项目是基于 SpringBoot 与 Vue 技术栈开发的社团管理系统，旨在为社团提供全面、便捷的管理功能。

## 技术栈

- 前端：Vue
- 后端：SpringBoot

## 功能模块

### 系统角色

系统中的用户角色分为以下三类：

- 系统管理员
- 社团管理员
- 普通用户

### 功能介绍

#### 系统管理员

- 首页查看
- 用户管理
- 社团管理
- 社团类型管理
- 社团成员管理
- 社团活动管理
- 通知信息管理
- 入团申请管理
- 费用管理
- 修改个人信息
- 修改密码

#### 社团管理员

- 首页查看
- 社团信息管理
- 社团成员管理
- 入团申请管理
- 社团活动管理
- 通知信息管理
- 费用管理
- 修改个人信息
- 修改密码

#### 用户

- 首页查看
- 社团信息查看
- 社团活动管理
- 提交入团申请
- 费用管理
- 修改个人信息
- 修改密码

## 运行环境

- 前端代码：VSCode 或其他支持 Vue 的编辑器
- 后端代码：IDEA 或其他支持 SpringBoot 的集成开发环境
